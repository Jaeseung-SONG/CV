# Jaeseung Song — Academic CV

Public academic profile and web CV for **Jaeseung Song, Ph.D.**, Research Assistant Professor at the Biomedical Research Institute, Seoul National University Hospital.

## View the website

After GitHub Pages is enabled, the site will be available at:

**https://jaeseung-song.github.io/CV_to_GitHub/**

## Publish with GitHub Pages

1. Create a public GitHub repository named `CV_to_GitHub`.
2. Push this folder to the repository.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`, then save.

No build process or external dependencies are required. The website is plain HTML, CSS, and JavaScript.

## Updating the CV

- Edit content in `index.html`.
- Edit colors and layout in `styles.css`.
- Update the date in the footer when publishing substantial changes.

## Privacy

The source Word CV, PDFs, and confidential references are intentionally excluded through `.gitignore`. Check staged files before every push.
